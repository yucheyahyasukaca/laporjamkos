# Changelog

Semua perubahan penting pada proyek ini akan didokumentasikan dalam file ini.

## [1.0.0] - 2026-02-07

### Fitur Baru
- **Dashboard Admin (Command Center)**: Desain ulang total dengan sapaan personal, ringkasan statistik, dan *live feed* aktivitas.
- **Halaman Laporan (Reports)**: Halaman khusus untuk memantau, memfilter, dan menyelesaikan laporan jam kosong.
- **Halaman Data Kelas (Classes)**: Manajemen kelas lengkap dengan fitur tambah, hapus, cetak QR Code, dan pencarian.
- **Toggle Tampilan**: Pilihan tampilan *Grid* atau *List* pada halaman Data Kelas (default *List*).
- **Footer Copyright**: Penambahan footer hak cipta yang konsisten di halaman Login dan Admin.

### Perbaikan
- Integrasi *backend* Supabase untuk semua fitur utama.
- Peningkatan UI/UX menggunakan *Glassmorphism* dan animasi halus.
- Perbaikan navigasi *sidebar* dan *routing*.
